Makefile é um tipo de arquivo que contém um conjunto de instruções para compilar e ligar um programa.

I2C é um protocolo utilizado para comunicar periféricos na placa mãe. Ele permite que dispositivos diferentes se comuniquem e troquem dados de forma rápida e eficiente. No caso da FPGA é utilizado para conectar com o acelerômetro. Sua maior vantagem é que pode ser utilizado para ligar múltiplos dispositivos através de um único barramento e também é versátil, podendo ser usado tanto em aplicações simples até sistemas de maior complexidade.

Na comunicação I2C, um dispositivos é designado como mestre e os demais como escravos. Cada dispositivo escravo é designado com um endereço próprio, de forma que o mestre sabe exatamente qual dispositivo deve ser acessado.
